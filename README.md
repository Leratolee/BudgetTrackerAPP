# BudgetTrackerAPP
Budget Tracker App

Overview

The Budget Tracker App is a mobile application developed using Kotlin in Android Studio. It is designed to help users manage their personal finances by recording income and expenses, making it easier to track spending and stay within a budget.
 Features

- User login screen for secure access
- Add income entries
- Add and track expenses
- View a simple financial summary of total income vs expenses
- Basic budget tracking functionality
- Clean and easy-to-use interface

 Technologies Used

-Kotlin
- Android Studio
- XML for designing layouts

 

Project Structure

- MainActivity.kt – Main dashboard of the app
- LoginActivity.kt – Handles user authentication
- activity_main.xml – Main UI layout
-activity_login.xml – Login screen layout
- res/ – Contains all resources such as layouts and images

Purpose

This project was developed as part of a learning assignment to demonstrate Android development skills, including user interface design, navigation between screens, and basic data handling
